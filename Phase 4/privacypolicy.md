## 1. Data Controller
This Booking System is operated for educational purposes.
The system owner acts as the Data Controller.

## 2. Personal Data Collected
- Email address
- Date of birth (age verification)
- User role (Reserver / Administrator)
- Reservation data (resource, time)

## 3. Purpose of Data Processing
Personal data is processed to:
- Provide booking functionality
- Verify user age
- Ensure system security
- Manage user accounts and reservations

## 4. Legal Basis for Processing
Data processing is based on:
- Performance of a contract (GDPR Art. 6(1)(b))
- Legal obligation to restrict underage usage

## 5. Data Retention
Personal data is stored only as long as the user account exists.
Administrators may delete accounts upon request.

## 6. Data Sharing
Personal data is not shared with third parties.
Data is used only within the Booking System.

## 7. User Rights
Users have the right to:
- Access their personal data
- Request deletion of their account
- Receive information about data processing

## 8. Data Security
Passwords are stored in hashed form.
Access control is enforced using role-based authorization.

## 9. Changes
This privacy policy may be updated when necessary.
